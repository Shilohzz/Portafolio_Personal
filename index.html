Gestor de Compras (Python)

Desarrollé un Gestor de Gastos Personales utilizando Python. El aplicativo permite a los usuarios registrar y clasificar gastos financieros con granularidad diaria, semanal, mensual y anual. Cada registro incluye una descripción detallada y la fecha de realización. Implementé persistencia de datos mediante el uso del formato JSON para asegurar el almacenamiento y la recuperación eficiente de la información.

Biografía

Soy un desarrollador Back-End junior apasionado por crear experiencias web excepcionales. Mi viaje en la programación me ha permitido contemplar paisajes hermosos de código, donde descubrí mi amor por resolver problemas complejos a través de la lógica.

Me especializo en el desarrollo backend con Node.js y MongoDB, pero también tengo experiencias prácticas en frontend con HTML y CSS (también algunos frameworks). Utilizo Git y GitHub en todos mis proyectos, siguiendo las mejores prácticas de control de versiones y colaboración en equipo para mantenerme siempre a la vanguardia del proyecto.

Cuando no estoy programando, me gusta contribuir a proyectos de código abierto, leer documentación técnica y mantenerme actualizado con las últimas tecnologías de la industria.
